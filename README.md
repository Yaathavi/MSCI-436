# MSCI436 - UniMatch Decision Support System

## Installation

To run this project on your local machine, follow these steps:

### Prerequisites

Ensure you have the following installed:

- Python 3.7 or higher
- `pip` (Python package installer)

### Steps

1. **Clone the Repository**

   ```bash
   git clone <repository-url>
   cd flask-server


2. **Create and Activate Virtual Environment**

   ```bash
   python3 -m venv venv
   source venv/bin/activate  # this is for mac 
   source venv\Scripts\activate` # On Windows use this 

3. **Install Dependencies**

   ```bash
   pip install -r requirements.txt


4. **Run the Application**

   ```bash
   python app.py

Now visit http://127.0.0.1:5000/ (the terminal will say what it's running on).
